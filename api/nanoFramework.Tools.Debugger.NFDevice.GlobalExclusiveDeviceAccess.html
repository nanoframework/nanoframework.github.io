Class GlobalExclusiveDeviceAccess

Namespace
nanoFramework.Tools.Debugger.NFDevice
Assembly
nanoFramework.Tools.DebugLibrary.Net.dll

Code that wants to access a device should use this system-wide exclusive access while communicating to a device to prevent that another nanoFramework tool also wants to communicate with the device. The public methods return an instance of GlobalExclusiveDeviceAccess if the access has been granted; that instance has to be disposed of when exclusive access is no longer needed.
