Severity represents the Severity in the Syslog messages. This corresponds to the values defined in RFC 3164 except None which means that the message won't be sent.

public enum Severity

Fields

Alert = 1

Alert: action must be taken immediately. No equivalent in LogLevel

Critical = 2

Critical: critical conditions. Mapped to Critical

Debug = 7

Debug: debug-level messages. Mapped to Debug and Trace

Emergency = 0

Emergency: system is unusable. No equivalent in LogLevel

Error = 3

Error: error conditions. Mapped to Error

Informational = 6

Informational: informational messages. Mapped to Information

None = -1

Not an official RFC3164 facility. No entry will be generated when called with this Severity

Notice = 5

Notice: normal but significant condition. No equivalent in LogLevel

Warning = 4

Warning: warning conditions. Mapped to Warning
