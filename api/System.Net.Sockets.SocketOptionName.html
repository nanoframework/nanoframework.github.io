    Enum SocketOptionName

    Defines socket option names for the Socket class.

    Namespace: System.Net.Sockets
    Assembly: System.Net.dll
    Syntax
    public enum SocketOptionName
    Remarks

    The SocketOptionName enumeration defines the name of each Socket configuration option. Sockets can be configured with the SetSocketOption(SocketOptionLevel, SocketOptionName, Boolean) method.
