    Remarks

    Segment representation:

    bit 0 = a a bit 1 = b | | bit 2 = c f b bit 3 = d |g| bit 4 = e | | bit 5 = f e c bit 6 = g |d| .dp bit 7 = dp

    Representation of the number 0 so lighting segments a, b, c, d, e and F is then 0x3f
