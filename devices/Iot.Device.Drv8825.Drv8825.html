Parameters

stepPin byte

Microcontroller pin connected to STEP driver pin. Used to set steps count.

dirPin byte

Microcontroller pin connected to DIR driver pin. Used to set rotation direction.

sleepPin byte

Microcontroller pin connected to SLP driver pin. Used to wake up and put the driver to sleep. Usually SLP need to connect with RST pin.

fullStepsPerRotation ushort

How many steps your motor need to make full rotation. For example, Nema 17 takes 200 full steps to complete full rotation.

gpioController GpioController

GPIO controller. If it not passed, then it be created here.

shouldDisposeGpioController bool

True to dispose the Gpio Controller when this class wiill be disposed.

m0Pin byte

Microcontroller pin connected to M0 driver pin. Can be used to microsteps control. 0 if not connected.

m1Pin byte

Microcontroller pin connected to M1 driver pin. Can be used to microsteps control. 0 if not connected.

m2Pin byte

StepDelay

Gets or sets the delay that allows the driver to recognize a step. By default, it is 5 microseconds. It is not recommended to set this delay less than 2 microseconds, otherwise the motor will "miss"(skip) some steps. And it is not recommended to set this delay more than 50 microseconds, otherwise the morot will stop recognize steps.
