Remarks

The serial communication of LD2410 uses little-endian format. All the values in this enum are in big-endian to match the manufacturer docs. Use System.Buffers.Binary.BinaryPrimitives.WriteUInt16LittleEndian(System.SpanByte, ushort) before writing to the serial port to ensure that the byte representation will be converted to little-endian if the code is running on a big-endian CPU.
