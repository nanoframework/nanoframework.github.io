    Remarks

Please refer to documentation to understand when to restore a previous baseline: https://www.sciosense.com/wp-content/uploads/2020/01/Application-Note-Baseline-Save-and-Restore-on-CCS811.pdf.

    Parameters
    Type Name Description
    VolumeConcentration equivalentCO2

    The equivalent CO2 (eCO2) output range for CCS811 is from 400ppm up to 29206ppm.
    VolumeConcentration equivalentTotalVolatileOrganicCompound

    The equivalent Total Volatile Organic Compound (eTVOC) output range for CCS811 is from 0ppb up to 32768ppb.
    ElectricCurrent rawCurrentSelected

    Raw data containing the value of the current through the sensor(0μA to 63μA).
    int rawAdcReading

    Raw data containing the readings of the voltage across the sensor with the selected current(1023 = 1.65V) where 1023 is the maximum value.
    Returns
    Type Description
    bool

    True if success.
