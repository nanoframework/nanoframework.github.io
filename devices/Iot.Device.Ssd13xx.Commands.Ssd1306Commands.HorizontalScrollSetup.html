    HorizontalScrollSetup(HorizontalScrollSetup.HorizontalScrollType, PageAddress, FrameFrequencyType, PageAddress)

    Initializes a new instance of the HorizontalScrollSetup class. This command consists of consecutive bytes to set up the horizontal scroll parameters and determines the scrolling start page, end page and scrolling speed.
