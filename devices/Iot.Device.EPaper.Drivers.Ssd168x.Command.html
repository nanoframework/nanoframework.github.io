    Enum Command

    Commands supported by SSD1681.

    Namespace: Iot.Device.EPaper.Drivers.Ssd168x
    Assembly: Iot.Device.ePaper.dll
    Syntax
    public enum Command : byte

    Fields

    Name Description
    AutoWriteRAMForRegularPatternBlackWhite

    Auto Write B/W RAM for regular pattern. The Busy pin will read high during this operation.
    AutoWriteRAMForRegularPatternRed

    Auto Write RED RAM for regular pattern. The Busy pin will read high during this operation.
    BoosterSoftStartControl

    Booster Enable with Phase 1, Phase 2 and Phase 3 for soft start current and duration setting.
    BorderWaveformControl

    Set Border Waveform Control values.
    CrcCalculation

    CRC Calculation Command. For information, refer to SSD1681 application note. The Busy pin will read high during this operation.
    CrcStatusRead

    CRC Status Read.
    DataEntryModeSetting

    Define data entry sequence.
    DeepSleepMode

    Set Deep Sleep Mode.
    DisplayUpdateControl1

    Display Update Control 1.
    DisplayUpdateControl2

    Display Update Sequence Option. Enables the stage for MasterActivation.
    DriverOutputControl

    Driver Output Control Command. Sets the gate, scanning order, etc.
    EndOption

    Option for LUT end.
    ExternalTempSensorControlWrite

    Temperature Sensor Control. Write command to External temperature sensor. The Busy pin will read high during this operation.
    GateDrivingVoltage

    Gate Driving Voltage Command.
    HvReadyDetection

    HV Ready Detection. The Busy pin will read high during this operation.
    InitialCodeSettingOtpProgram

    Program Initial Code Setting.
    LoadWSOTP

    Load OTP of Waveform Setting. The Busy pin will read high during this operation.
    MasterActivation

    Master Activation. Activate display update sequence. The Busy pin will read high during this operation.
    NOP

    NOP. This command is an empty command; it does not have any effect on the display module. However, it can be used to terminate frame memory write or read commands.
    OTPProgramMode

    OTP Program Mode.
    OTPRegisterReadDisplayOption

    OTP Register Read for Display Options.
    ProgramOTPSelection

    Program OTP Selection according to the WriteRegisterForDisplayOption and WriteRegisterForUserId. The Busy pin will read high during this operation.
    ProgramVCOMOTP

    Program VCOM register into OTP. The Busy pin will read high during this operation.
    ProgramWSOTP

    Program Waveform Setting OTP. Contents should be written to RAM before sending this command. The Busy pin will read high during this operation.
    ReadRAM

    Read RAM. After this command, data read on the MCU bus will fetch data from RAM. The first byte read is dummy data.
    ReadRAMOption

    Read RAM Option.
    ReadRegisterForInitialCodeSetting

    Read Register for Initial Code Setting.
    SetRAMAddressCounterX

    Set RAM X-Address Counter.
    SetRAMAddressCounterY

    Set RAM Y-Address Counter.
    SetRAMAddressXStartEndPosition

    Set RAM X-Address Start/End position.
    SetRAMAddressYStartEndPosition

    Set RAM Y-Address Start/End position.
    SoftwareReset

    Software Reset. This resets the commands and parameters to their S/W default values except Deep Sleep Mode. The Busy pin will read high during this operation. RAM contents are not affected by this command.
    SourceDrivingVoltageControl

    Source Driving Voltage Control.
    StatusBitRead

    Read Status Bit.
    TempSensorControlReadRegister

    Read from temperature register.
    TempSensorControlSelection

    Temperature Sensor Selection (External vs Internal).
    TempSensorControlWriteRegister

    Write to temperature register.
    UserIdRead

    Read USER ID.
    VciDetection

    VCI Detection. The Busy pin will read high during this operation.
    VCOMSense

    Enter VCOM sensing conditions. The Busy pin will read high during this operation.
    VCOMSenseDuration

    VCOM Sense Duration. Stabling time between entering VCOM sending mode and reading is acquired.
    WriteBackWhiteRAM

    Write To B/W RAM. After this command, data will be written to the B/W RAM until another command is sent. Address pointers will advance accordingly.
    WriteLUTRegister

    Write LUT Register from MCU interface.
    WriteRedRAM

    Write To RED RAM. After this command, data will be written to the RED RAM until another command is sent. Address pointers will advance accordingly.
    WriteRegisterControlVCOM

    Write Register for VCOM Control. This command is used to reduce glitch when ACVCOM toggle.
    WriteRegisterForDisplayOption

    Write Register for Display Option.
    WriteRegisterForInitialCodeSetting

    Write Register for Initial Code Setting Selection.
    WriteRegisterForUserId

    Write Register for USER ID.
    WriteVCOMRegister

    Write VCOM register from MCU interface.
