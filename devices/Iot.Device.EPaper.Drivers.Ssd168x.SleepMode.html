    Enum SleepMode

    SSD1681 Supported Sleep Modes.

    Namespace: Iot.Device.EPaper.Drivers.Ssd168x
    Assembly: Iot.Device.ePaper.dll
    Syntax
    public enum SleepMode : byte

    Fields

    Name Description
    DeepSleepModeOne

    Deep Sleep Mode 1. In this mode:

    • DC/DC Off
    • No Clock
    • No Output load
    • MCU Interface Access: OFF
    • RAM Data Access: ON (RAM contents retained).
    DeepSleepModeTwo

    Deep Sleep Mode 2. In this mode:

    • DC/DC Off
    • No Clock
    • No Output load
    • MCU Interface Access: OFF
    • RAM Data Access: OFF (RAM contents NOT retained).
    Normal

    Normal Sleep Mode. In this mode:

    • DC/DC Off
    • No Clock
    • No Output load
    • MCU Interface Access: ON
    • RAM Data Access: ON.
