Radar(string, int, TimeSpan)

Parameters

serialPortName string

The serial port name to use.

baudRate int

The baud-rate to use.

commandTimeout TimeSpan

The command timeout. If not specified, the default is 5 seconds.

Remarks

Initializing the Radar is not enough to connect to the radar. You must call Connect() when ready to start reading measurements and controlling the radar.

Exceptions

ArgumentNullException

serialPortName was null or empty string.

ArgumentOutOfRangeException

baudRate was equal or less than 0.

SetEngineeringMode(bool)

Enables or disables the engineering mode on the radar.

public void SetEngineeringMode(bool enabled)

Parameters

enabled bool

true to enable engineering mode.

Remarks

Enabling engineering mode makes the radar append additional data to every measurement frame. This data will include energy measurements at every gate for moving and static targets.
