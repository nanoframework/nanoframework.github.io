SetSegmentReMap(bool)

Initializes a new instance of the SetSegmentReMap class. This command changes the mapping between the display data column address and the segment driver. It allows flexibility in OLED module design. This command only affects subsequent data input. Data already stored in GDDRAM will have no changes.
