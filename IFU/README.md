# IFU (In-Field Update)

This folder contains the in-field update (IFU) specification documents for .NET nanoFramework. These documents describe the architecture and implementation path for adding production-grade update workflows, along with a concise executive summary for stakeholders.

Implementation is currently underway. Reach out on the Discord channel (https://discord.gg/JnE2BnWj) if you want to contribute or join the conversation.

## Documents

- [Executive Summary](Executive-Summary.md) - High-level goals, approach, scope, and effort estimates.
- [In-field update for nanoFramework](In-field-Update-design.md) - Detailed design notes, architecture, workflows, and FAQ.
