# IFU (In-Field Update)

This folder contains the in-field update (IFU) proposal documents for .NET nanoFramework. The intent is to capture the vision, architecture, and implementation path for adding production-grade update workflows, along with a concise executive summary for stakeholders.

This is currently an ongoing effort. We are looking for sponsors to cover the development cost. Take a look at the executive summary and/or reach out on the Discord channel (https://discord.gg/JnE2BnWj) if you want to contribute or join the conversation.
